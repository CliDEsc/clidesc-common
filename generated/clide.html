clide

clide.py provides helper functions to support the extraction of data from a CliDE database.

Author:

James Sturman, NIWA

Date:

December 2022

Usage:

import this module into your Python script and call the functions as needed.

Note

When returning a table (e.g., the result of a SQL query to the clide database), this table is a Pandas DataFrame object, which is similar to (but more efficient than) an R dataframe, and makes slicing, column or row selection, resampling, etc. extremely convenient. When time-series are returned, the index of the dataframe (i.e., the ‘rows’ identifier) is a Pandas DatetimeIndex object resulting from the conversion of the lsd field to a Python datetime object and setting it as the index for the DataFrame. The name of the index is invariably ‘timestamp’.
